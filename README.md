## Run Locally

Clone the project

```bash
  git clone https://github.com/KrishanSharma/SharmaStoreNextWebsite.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
